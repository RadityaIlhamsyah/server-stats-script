# server-stats-script

Skrip Bash sederhana untuk menganalisis statistik kinerja server Linux secara instan.

## Fitur
- Total Penggunaan CPU.
- Penggunaan Memori (Used vs Free dengan persentase).
- Penggunaan Disk (Used vs Free dengan persentase).
- 5 Proses teratas berdasarkan penggunaan CPU & Memori.
- Informasi tambahan: OS Version, Uptime, dan Load Average.
